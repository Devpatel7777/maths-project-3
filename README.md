# Derivable Judgement: A Statistical Decision-Making Model

## Overview

This project focuses on applying inferential statistics and hypothesis testing techniques to analyze healthcare-related data. The analysis helps derive meaningful judgments and conclusions from the dataset using statistical methods and visualizations.

The project demonstrates practical implementation of statistical concepts such as confidence intervals, z-test, t-test, chi-square test, ANOVA, covariance, and correlation using Python.

# Statistical Techniques Used

## 1. Confidence Interval

Used to estimate the range within which the population parameter lies.

## 2. z-test

Used to compare sample mean with population mean.

## 3. t-test

Used to compare BMI between male and female groups.

## 4. Chi-square Test

Used to check association between smoking status and diabetes.

## 5. ANOVA Test

Used to compare BMI across multiple age groups.

## 6. Covariance

Measures how two variables vary together.

## 7. Correlation

Measures strength and direction of relationship between variables.
